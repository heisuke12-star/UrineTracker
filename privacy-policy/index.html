개인정보처리방침

앱명: 소변 트래커  |  시행일: 2026년 5월 27일

한 줄 요약: 소변 트래커는 모든 데이터를 사용자의 기기에만 저장합니다. 어떠한 개인정보도 수집하거나 외부로 전송하지 않습니다.

1. 수집하는 정보

소변 트래커는 다음 데이터를 사용자의 기기 내부에만 저장합니다.

  • 소변 및 수분 섭취 기록 (날짜, 시간, 양, 종류)
  • 앱 설정값 (경고 알림 여부, 최소·최대 정상 수치)

이 데이터는 앱의 기능 제공 목적으로만 사용되며, 개발자 또는 제3자에게 전송되지 않습니다.

2. 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 식별 가능한 개인정보
  • 위치 정보
  • 기기 고유 식별자
  • 사용 통계 또는 분석 데이터

3. 제3자 정보 공유

소변 트래커는 어떠한 제3자와도 사용자 데이터를 공유하지 않습니다. 광고 SDK, 분석 도구, 외부 서버를 포함하지 않습니다.

4. 데이터 저장 및 삭제

모든 데이터는 Android의 SharedPreferences에 기기 로컬로 저장됩니다. 데이터 삭제 방법은 다음과 같습니다.

  • 개별 삭제: 기록을 길게 눌러 삭제
  • 전체 삭제: 설정 화면 → 전체 기록 삭제
  • 앱 삭제: 앱을 삭제하면 모든 데이터가 함께 삭제됩니다

5. 어린이 개인정보 보호

본 앱은 만 14세 미만 어린이를 대상으로 하지 않으며, 어린이의 개인정보를 의도적으로 수집하지 않습니다.

6. 방침 변경

개인정보처리방침이 변경될 경우 이 페이지를 통해 공지합니다. 변경 사항은 이 페이지에 게시된 시점부터 효력이 발생합니다.
